# foundry-cloudflared

My simple [FoundryVTT](https://foundryvtt.com) + [Cloudflare Tunnel](https://www.cloudflare.com/products/tunnel/) setup

## Environment Variables

### env.d/cloudflared
```
TUNNEL_TOKEN=
```

### env.d/foundry
```
FOUNDRY_PASSWORD=
FOUNDRY_USERNAME=
FOUNDRY_ADMIN_KEY=
CONTAINER_PRESERVE_CONFIG=true
```
